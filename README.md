# Cartoon_gllmh
鬼萝丽漫画自动下载

网站地址：
http://www.gllmh.com
使用方法：
①：打开网站后选择需要下载的漫画
②：在选择的漫画首页复制完整地址（例如：http://www.gllmh.com/jqsz/6890.html ）
③：漫画自动下载至与源文件同级目录的image文件夹中

Ps：下载的内容供个人使用，请勿传播。
    需要下载多部漫画时请及时移走image文件夹中已下载完的漫画避免被新下载的漫画所覆盖。
    部分漫画宽度并非750如无法下载请使用开发人员工具查看漫画图片img标签属性中的data-width的值并修改程序中的预设值。
